◆ updateTimeIntegral()

int CgWave::updateTimeIntegral ( int  step,
StepOptionEnum  stepOption,
real  t,
realCompositeGridFunction &  u 
)

Update the time integral used by the Helmholtz solver.

The WaveHoltz algorithm computes the time integral of the solution. This routine increments the time integral using the current solution and stores the result in "v" (from the dbase).

Parameters
step(input) : current step
stepOption: firstStep, middleStep, lastStep. When stepOption==firstStep, the integration weights will be comnputed and the time integral will be initialized.
t(input) : current time
u(input) : current solution
